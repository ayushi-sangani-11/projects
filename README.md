# 👟 UI/UX Portfolio – Shoe App & Product Design Collection

## 📌 Overview
This project showcases my UI/UX design work featuring wireframes and interface concepts created for product and shopping experiences. The designs focus on creating visually engaging, user-friendly, and modern mobile/web interfaces while improving usability and aesthetics.

This portfolio demonstrates my design thinking process, visual design skills, and ability to create intuitive user experiences.

---

## 🎯 Project Goal

The main goal of this project was to:

- Design engaging and modern user interfaces
- Create visually appealing shopping experiences
- Improve usability and navigation
- Explore different design styles and layouts
- Practice UI principles and mobile-first thinking

---

## 🖼 Featured Designs

### 1. Juice Landing Page (Web Design)
A modern beverage landing page concept with a bold visual hierarchy.

**Features:**
- Large hero section
- Product-focused design
- Clean navigation bar
- Interactive bottle-size options
- Strong typography and branding
- Scroll interaction element

**Design Focus:**
Creating an attractive landing page that instantly grabs user attention.

---

### 2. Nike Shoe Product Detail Screen

A product detail page designed for a shopping experience.

**Features:**
- Product image spotlight
- Quantity selector (+ / -)
- Product pricing
- Ratings display
- Shoe size selector
- Add Basket & Shop buttons

**Design Focus:**
Making product purchasing easy with clear information hierarchy.

---

### 3. Nike Shopping Home Screen

A mobile shopping home interface with categorized products.

**Features:**
- Promotional banner
- Category chips
- Product cards
- Bottom navigation
- Lifestyle, Sports, Running categories
- Minimal and clean layout

**Design Focus:**
Building a seamless browsing experience.

---

## 🎨 Design Process

### 1. Research
- Studied shopping and product UI inspirations
- Explored modern ecommerce trends
- Identified user needs

### 2. Wireframing
Created low-fidelity structures to plan:

- Screen hierarchy
- Navigation flow
- Component placement

### 3. Visual Design
Applied:

- Typography hierarchy
- Color systems
- Spacing principles
- Visual consistency

### 4. Iteration
Refined layouts to improve usability and aesthetics.
